# パターン集

スケーラブルオーバーサイトの文脈で観察された、再利用可能な設計パターン。

各パターンは以下の形式で記述する：

- **意図** — 何の問題を解くか
- **動機** — なぜそのパターンが必要か
- **構造** — どのように構成されるか
- **帰結** — 得られるもの・失うもの
- **既知の使用例**
- **関連パターン**
