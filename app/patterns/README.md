# パターン集

スケーラブルオーバーサイトの文脈で観察された、再利用可能な設計パターン。

各パターンは WikiXP（Portland Pattern Repository）形式で記述される：

- **コンテキスト** — このパターンが適用される状況
- **問題** — 解くべき問題（太字で記述）
- **フォース** — 問題を難しくする競合する制約
- **解決策** — パターンの解決策（太字で記述）
- **結果コンテキスト** — 適用後の状況（得られるもの・失うもの）
- **既知の使用例** — 実世界での適用事例
- **関連パターン** — 他パターンへのリンク
- **ソース** — 根拠となった知識ファイルへのリンク
