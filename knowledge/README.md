## 追加ガイドライン

新しいナレッジを追加する際は：
1. ソース URL と追加日を明記する
2. 「スケーラブルオーバーサイトへの示唆」セクションでパターンとの接続を記述する
3. 既存パターンと一致する場合はパターンファイルの「既知の使用例」に追記する
4. 新しいパターンを発見した場合は `patterns/` に新規ファイルを作成する
